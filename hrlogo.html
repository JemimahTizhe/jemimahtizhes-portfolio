HR Logo Design (Practice Work)

Project Type: Personal Practice

Design Inspired By: YouTube Tutorial

Date: January 2024

Project Overview:

This logo was created as part of a practice session to improve my 3D logo design skills using tools like Adobe Illustrator and Photoshop. The initials “HR” are placed within a circular layout, accented by a diagonal line to create a sense of motion and strength.

Key Elements Practiced:

  • Typography arrangement and logo symmetry
  • Circular layout and text on path
  • 3D embossing effect and realistic rendering
  • Creating contrast using gold and black color palettes
  • Lighting and shadow realism in 3D logos

Tools Used:

  • Adobe Illustrator (Logo structure)
  • Adobe Photoshop / Blender (for 3D look and rendering)

What I Learned:

This project helped me sharpen my understanding of logo composition, realistic lighting for 3D effects, and professional logo presentation. Although inspired by a tutorial, I customized the text and details to fit my personal brand.
